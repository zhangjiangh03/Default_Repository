凯里学院青年志愿者联合会

图管部 一支志愿者队伍，我们践行志愿者精神，不为物质报酬，基于良知、信念和责任，自愿为社会和他人提供服务和帮助。立足图书馆，无偿服务。使“奉献、友爱、互助、进步”的青年志愿者精神立足在青年一代中发扬光大。
实践部 其部门职能主要有：根据制定好的工作方案组织志愿者开展志愿服务活动；做好活动前期准备工作；负责申请或借用各种物资、场地，活动开展的踩点、跟进及后期总结。
宣传部 负责联合会工作的宣传，加强与党委宣传部、团委的联系。做好青联工作动态的宣传报道，对各项志愿服务活动通过新型媒体及时更新、加大宣传力度。

策划部 根据校团委有关工作要求与联合会的意志拟定联合会的工作计划；负责联合会志愿贵州和到梦空间的管理以及各类活动的策划；负责凯里学院卫生知识传播志愿队的指导工作，并统筹规划卫生知识传播志愿队的发展；为联合会争取社会各方面的支持，起草所有活动方案。

凯里学院学生会

凯里学院学生会成立于1978年。校学生会一直在学校党委的领导下,在校团委的具休指导下开展全校学生工作的群#性组织。学生会下设主席团、办公室、室传部、权益部、学习部、文休部、实践部，其工作范围涉及全校学生的学习、生活、权益等各个方面、各个领域。历年来,在省校两级校园文化活动月以及各类大型活动和晚会中发挥重要作用，承办经验丰富。形成了“求真、务实、高效、创新”的工作作风。

凯里学院第二课堂

“到梦空间”系统，又称为高校共青团“第二课堂成绩单”网络管理系统,于2017年9月在我校正式上线运行。学校借助“到梦空间”系统可以对第二课堂活动进行模块划分、信息发布、过程管理和效果评价可以对发起第二课堂活动的团学组织进行管理、监督考核和评价，可以对学生参与第二课堂活动进行记录、评价和认证。该系统的大数据分析功能将为学校人才培养和第二课堂工作提供咨询诊断、具体建议和决策参考。

凯里学院大学生通讯社

本社于2004年成立，隶属于凯里学院党委宣传部，是由宣传科负责日常管理和指导,以培养新闻文学人才,引导学生社会实践为指导思想;以凯里学院官网主页、凯里学院报、凯里学院官方微信为主要工作平台的校级学生组织。

分院组织

学校分院组织是由各个分院学生组成的学生组织，是校级组织的分支下属，常有院青联、院学生会、院二课等。

学生社团

学生社团是指由学生自发组织和管理的、具有特定目标和活动内容的、以提高学生综合素质和促进校园文化建设为宗旨的非营利性团体。学生社团通常包括学术性社团、文化艺术类社团、体育健身社团、公益志愿团体、科技创新团队等不同类型。

春晖行动服务团

凯里学院春晖行动服务团成立于2009年,原名凯里学院大学生春晖社、社团春晖行动小组,2013年4月14日正式更名为凯里学院春晖行动服务团,是由校党委领导,校团委直接指导下的学生组织。下设理事会、秘书处、支教队、文艺宣讲队、维修队、后勤队六大机构部门。始终以弘扬“饮水思源、反哺故土、回报社会”为宗旨,以“弘扬中华文明,反哺故土亲人”为最高宗旨,以“尽孝、感恩、反哺、回报”为理念。自从组建以来曾到过麻江县龙山乡、凯里市格河村、翁义小学、老校区、棉纺厂等多地开展涉及义务支教、家电维修、思想理念宣传、文艺等大小几十次公益活动,多次荣获各级奖项,于2013年10月在翁义小学签约成立了凯里学院春晖行动社会实践基地。
