大学生艺术团

凯里学院大学生艺术团(简称“艺术团”)是在凯里学院校团委领导下,表演与管理结合,由具有一定艺术专长和组织能力的同学组成的一个具有特色的校级大学生艺术团体。

吉他协会

凯里学院吉他协会于2018年9月份创建，挂靠于音乐与舞蹈学院,指导老师熊毅老师。在熊毅老师的指导下，我们协会很快就展开了各种类型的活动，其中包括教学，演出。

金谷圆棋社

金谷圆棋社成立于2010年,主要涉及棋类:五子棋，中国象棋,围棋,国际象棋,在这里,选拔出了一支凯里学院中国象棋校队棋社宗旨:以棋会友，乐在棋中。

凯里学院春晖行动服务团

凯里学院春晖行动服务团成立于2009年,原凯里学院大学生春晖社社团春晖行动小组,13年4月14日更名为凯里学院春晖行动服务团,是由校党委领导，校团委直接指导下的学生组织。
